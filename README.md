Since this is a <b>website</b> you can look at it by going onto the link:

https://website-workshop-seven.vercel.app/

- Home page
- Basic Html/CSS website guide + deployment guide

- Mac set up guide
- Linux set up guide

- Basic Django website guide + deployment guide
